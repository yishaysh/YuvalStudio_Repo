קופה מהירה חדש

מכרת ביריד? עמד מישהו בתור בדוכן? עכשיו אפשר לתעד כל מכירה ישירות, בלי להמציא תור פיקטיבי.

  • כפתור "קופה מהירה ⚡" מופיע בפינה השמאלית התחתונה בכל מסכי ה-Admin — תמיד זמין.
  • בוחרים שירות ו/או תכשיטים מהמלאי, המחיר והרווח מחושבים אוטומטית.
  • אפשר לשמור שם וטלפון לקוח — לא חובה. ברירת המחדל: "לקוח מזדמן".
  • לחיצה על "סיים וגבה תשלום" → המכירה נשמרת מיד כ-הושלם ומתעדכנת בסטטיסטיקות ובדוחות.
  • אין יותר צורך ביצירת תורים ידניים! 🎉

מערכת התראות חדש

טאב "התראות" חדש בממשק הניהול — קבלי התראה מיידית בכל פעם שלקוחה קובעת תור. שלוש שיטות, מתג עצמאי לכל אחת.

💬 Telegram Bot (חינם לגמרי)

  • הודעה מיידית לטלגרם עם שם הלקוחה, שירות, תאריך ושעה.
  • הגדרה: צרי בוט ב-@BotFather, הכניסי Bot Token + Chat ID בטאב "התראות".

📧 Email (חינם עד 200 הודעות/חודש)

  • אימייל אוטומטי לכתובת שתבחרי — נוחת בתוך שניות מקביעת התור.
  • הגדרה: הירשמי ל-emailjs.com, צרי Template עם שדות הלקוח, הכניסי Service ID + Template ID + Public Key.

📅 סנכרון גוגל יומן אוטומטי

  • כל תור חדש נוחת ביומן גוגל ברגע הקביעה — ללא צורך בפעולה ידנית.
  • הגדרה: הפעלי את המתג — הסנכרון קורה דרך חשבון הגוגל המחובר ל-Admin.
